## Staring up application

**pharmacy-app**

 * npm install
 * npm start
 
**pharmacyapp**

 - from console or bash
   - mvn clean install
   - move to pharmacypp/target folder
   - java -jar pharmacyapp-0.0.1-SNAPSHOT
  
* run application from IDE
